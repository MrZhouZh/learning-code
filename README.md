## Quick Start

```sh
# install
pnpm i

# 测试
pnpm test

# or
pnpm test -- [目录路径]

# 测试覆盖率
pnpm start

# promise 编译
pnpm build:jpromise

# promise 测试
pnpm test:promise [promise文件路径]
```
